# java-matopeli2009

Artistojen kätköistä löytynyttä. About 2009  kehitetty matopeli. Modernisoitu 2025 appletista Swing sovellukseksi toimimaan Javan JDK21 ympäristössä. Suottaapi olla alkuperältään parisen vuotta 2009 aikaisemmalta ajalta.

- Uusi peli alkaa aina painamalle space näppäintä.
- Peli menee pauselle hiirenklikkauksesta peli alueelle. 

Kehitettävää:
- Epävakaasti toimi pause.
- Eriytä pause "game over" tilasta
